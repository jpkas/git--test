Html-Css-Javascript Web Page Portofolio.

Power of div,id and class(es).

By JeanPierre Kasonga

This is html-css-javascript web page that I designed with the simple desire to show how master these technology. That said don't try to find in this page the responsive features for Mobile first.When I will build a Bootstrap page you will see that.This is an exercise for beginner web developers to see what can be done with html,css and javascript. So ,I have chosen div(s) to hold main elements of this page ,namelly an ordered list,two images,an unordered list, a table and a form. Without CSS the page is vertical each element following the other without colors and movement.This html structure is like the human skeleton without flesh which is of course scury or simply boring.To embelish the page I created a file index.css in the same folder where I created the index.html and the two images. With the CSS- display-flex property applied to classes embeded in div(s) , I was able to obtain an horizontal layout for buttons,lists and images.Then I applied colors and background-colors to elements,id(s) and classes(again) selected for this purpose. Now the page is looking pretty good like a human body. However it doesn't have movement like a monument, no interactivity. With Javascript code embeded in the script element that I placed before the body closing tag I gave motion to the page. I linked 4 buttons to the id of each element to be toggled. With the magic of javascript addEventListener code the computer is able to listen to the click upon the button leading to making the specified element appear or disappear from the page.It is magic, isn't ? By clicking on these buttons one after another you will remove from the page the student items list, the Washington obelisk monument and people in St Theresa church images, and, finally, the table of fruits with their prices. You will leave on the page only the buttons,texts(intro and conclusion) and the form to be filled. Vis-versa if you want you will return them again just by clicking those buttons.I hope you will be pleased to fill the form with your credentials and submit.
